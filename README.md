
# Employee Management System (JDBC + MySQL)
A simple CRUD application using Java, JDBC, and MySQL.

## Features
- Add Employees
- View Employees
- Remove Employees
- Count Employees

## Technologies Used
- Java
- JDBC
- MySQL

## How to Run
1. Configure MySQL and create the `Employee` table.
2. Update database credentials in `Impliment.java`.
3. Run `MainApp.java`.
